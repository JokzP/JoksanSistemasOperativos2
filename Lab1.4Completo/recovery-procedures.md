# Recovery Procedures

Este documento resume los procedimientos generales de recuperación explicados con más detalle en otros archivos:

1. Uso de GRUB Rescue Mode para entrar al modo normal de GRUB.
2. Arranque manual del sistema desde GRUB.
3. Recuperación desde un Live USB con montaje de particiones y uso de chroot.
4. Reinstalación de GRUB en el disco principal.
5. Restauración del archivo `/etc/default/grub` desde un respaldo.
6. Regeneración de `grub.cfg` con `update-grub`.
